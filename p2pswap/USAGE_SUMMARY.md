# 使用总结

## 快速开始

### 1. 启动开发服务器

```bash
cd p2pswap/frontend
npm install
npm run dev
```

访问：http://localhost:5173/designer

### 2. 配置地址池（500-1000个地址）

在 Designer 页面中：
1. 在"低风险地址池管理"中添加地址
2. 支持单个添加或批量导入
3. 建议至少 500 个地址

### 3. 输入高风险地址

在"高风险地址列表"中：
- 输入格式：`地址,金额1,金额2,金额3`
- 示例：`0x1234...,1000,2000,1500`
- 表示该地址需要处理 3 笔交易

### 4. 生成策略

1. 配置策略参数（可选）
2. 点击"生成操作计划"
3. 系统自动：
   - 使用高风险地址生成高风险交易
   - 从地址池选择地址生成正常交易
   - 应用所有防匹配策略

### 5. 自动执行任务

1. 切换到"任务列表"标签页
2. 等待任务就绪（系统自动监控）
3. 点击"执行"按钮，使用 Signer 自动签名执行

## 核心功能

✅ **地址池管理**：500-1000 个低风险地址
✅ **高风险地址输入**：简单格式，逗号分隔金额
✅ **自动策略生成**：系统自动选择地址和应用策略
✅ **Signer 自动签名**：无需用户交互，直接执行

## 访问地址

- Designer 页面：http://localhost:5173/designer
- 预处理页面：http://localhost:5173/preprocess
